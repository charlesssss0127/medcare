# MedCare · 長者食藥記錄 App

一個俾工人姐姐記錄屋企老人家食藥情況嘅 iPad Web App（PWA）。每次食藥都要影相為證，家人可以隨時翻查，用透明記錄化解信任問題。

- **英文為主 + 中文對照**，大字體、專業簡潔介面
- **家人用 4 位 PIN** 管理藥物同時間表；**姐姐只可以記錄，記錄一經確認唔可以刪改**
- 定時鬧鐘提示食藥；可延遲（15/30/60 分鐘），跳過必須揀原因
- 完全離線運行，資料只存喺 iPad 本機（IndexedDB）
- 冇需要 Mac、冇需要 App Store、冇月費

---

## 一、部署上 GitHub Pages（一次過，約 5 分鐘）

PWA 要行喺 HTTPS 先影到相同加到主畫面。最簡單就用免費嘅 GitHub Pages。

1. 喺 GitHub 建立一個新 repository（例如 `medcare`），設為 **Public**。
2. 將 `medication-tracker/` 資料夾入面**全部檔案**（`index.html`、`app.js`、`styles.css`、`manifest.json`、`sw.js`、`icons/`）上載到 repo 根目錄。
3. 入 repo → **Settings** → **Pages** → Source 揀 `main` branch、`/ (root)` → **Save**。
4. 等一兩分鐘，會出現網址：`https://<你嘅帳戶>.github.io/medcare/`。
5. 喺 iPad Safari 打開呢個網址。

> 唔想用 GitHub 都可以放喺任何支援 HTTPS 嘅靜態網站主機（Netlify、Cloudflare Pages 等），做法一樣：上載成個資料夾就得。

---

## 二、加到 iPad 主畫面（變成 App）

1. 用 **Safari**（一定要 Safari，唔可以 Chrome）打開上面個網址。
2. 撳底部（或右上角）嘅**分享**按鈕 <kbd>􀈂</kbd>。
3. 揀 **加入主畫面 (Add to Home Screen)** → **加入**。
4. 主畫面會出現「MedCare」圖示。以後撳圖示會**全螢幕**開啟，冇 Safari 網址列，同真 App 一樣。

第一次開之後，App 會自己 cache 起，**之後冇網都用得**。

---

## 三、第一次設定

第一次開會入**設定畫面**：

1. 輸入老人家名（中英文）同影一張相。
2. 設定 **4 位家人 PIN 密碼**（記住佢！之後改藥物／時間表都要用）。
3. 撳「開始使用」——會自動加入幾隻示範藥物同時間表方便你試。
4. 入「管理 · Manage」分頁（要 PIN）就可以改藥物、劑量、影每隻藥丸嘅相、設定食藥時間表。

### 藥物已經預先執好？（分格藥盒）

如果屋企人已經將藥分格執好落藥盒（唔使姐姐逐隻執），加時段嗰陣開啟 **「📦 Medicine already prepared · 藥物已準備好」** 個掣就得，唔使揀個別藥物。到時姐姐只需喺 app 確認「食咗準備好嘅藥」+ 影相，一樣有齊記錄同鬧鐘提示。想列明每隻藥就唔好開呢個掣，照揀藥物即可 —— 兩種時段可以並存。

## 四、令 App 長開唔熄（重要 — 令鬧鐘提示準時響）

iOS 嘅 Web App **唔支援背景推送通知**。所以呢個 App 設計成**開住喺 iPad 度**做鬧鐘：到食藥時間會全螢幕彈出 + 響鬧鐘。要準時響，App 要一直開住：

1. **設定 → 螢幕與亮度 → 自動鎖定 → 永不 (Never)**，令 iPad 唔會自動熄屏。
2. （建議）用一部固定放喺客廳／老人家房嘅 iPad，插住電，開住 MedCare。
3. **（進階）用「引導使用模式 Guided Access」鎖住 App**，防止老人家／小朋友撳出去：
   - 設定 → 輔助使用 → 引導使用模式 → 開啟，設一個密碼。
   - 開住 MedCare，快速撳 **3 下側邊鍵**（或主畫面鍵）→ 撳「開始」。
   - 咁樣 iPad 就會鎖死喺呢個 App，要密碼先出到去。

## 五、日常使用（工人姐姐）

- **今日 · Today**：睇今日每個時段要食咩藥。到鐘會全螢幕響鬧鐘。
- 撳時段卡（或鬧鐘畫面「記錄食藥」）：
  1. 逐隻藥剔一剔（對住藥丸相執藥）。
  2. 撳 **✓ 而家食** → **影一張老人家食藥嘅相**（必須）→ 確認。記錄即刻鎖定。
  3. 老人家想遲啲食 → **⏰ 延遲**，揀 15/30/60 分鐘，時間到會再響。
  4. 老人家唔肯食 → **✗ 跳過**，**必須揀原因**（拒絕／唔舒服／瞓咗／唔喺屋企／其他）先閂到提示。
- **記錄 · History**：家人睇月曆（綠＝全部食晒／黃＝有遲／紅＝有跳過），撳入去睇每次食藥嘅相同原因，仲有 7 日／30 日服藥率。

## 六、備份資料（強烈建議定期做）

資料只存喺呢部 iPad。如果 iPad 壞咗或者被清 Safari 資料，記錄會冇。所以：

- 入 **管理 → 匯出備份**，會下載一個 JSON 檔（連相片），儲落 iCloud／電腦／email 俾自己。
- 建議每星期／每月做一次。

---

## 技術備註

- 純前端：`index.html` + `app.js` + `styles.css`，冇 build step、冇 server。
- 資料庫：IndexedDB（藥物、時間表、記錄、相片 blob）。相片自動壓縮至 ~1280px JPEG。
- 離線：service worker（`sw.js`）cache 晒 app shell。
- 鬧鐘聲：Web Audio；PIN 用 SHA-256 hash 儲存。
- 記錄設計為 immutable：一經確認，介面冇任何刪改入口，每條記錄有建立時間戳。

## 疑難排解

| 問題 | 解決 |
|---|---|
| 影相變咗揀相簿，唔開相機 | iPad Safari 先會直接開相機；電腦瀏覽器會變揀檔。實機測試就正常。 |
| 冇聲響 | iOS 要用戶撳過螢幕一次先俾出聲。開 App 後撳一下畫面即可；並確認 iPad 冇校靜音／調高音量。 |
| 加唔到主畫面 | 一定要用 **Safari**，唔可以用 Chrome/Firefox。 |
| 資料唔見咗 | 唔好用無痕模式；唔好喺 Safari 設定「清除網站資料」。定期匯出備份。 |
